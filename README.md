# Калькулятор с памятью

Проект представляет собой классическое приложение-калькулятор с функцией памяти. Реализован на платформе WPF и .NET Framework 4.8.

## Функциональность

- Базовые арифметические операции: сложение, вычитание, умножение, деление.
- Поддержка десятичных чисел.
- Функция памяти:
  - `M+` — добавить текущее значение к содержимому памяти.
  - `M-` — вычесть текущее значение из содержимого памяти.
  - `MR` — восстановить число из памяти.
  - `MC` — очистить память.
- Сброс текущего ввода (`C`).
- Изменение знака числа (`+/-`).

## Технические детали

- **Язык:** C#
- **Фреймворк:** .NET Framework 4.8
- **UI:** Windows Presentation Foundation (WPF)
- **Стиль:** XAML без внешних библиотек

## Запуск

1. Установите .NET Framework 4.8.
2. Откройте решение в Visual Studio 2019 или новее.
3. Соберите проект (Build → Build Solution).
4. Запустите приложение.
